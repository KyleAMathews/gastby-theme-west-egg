# gatsby-theme-west-egg

A Gatsby theme package built with Emotion and Typescript consisting of:

- Style theme
- Blog theme
- Cart theme
- Master theme
- Example site

Current status: WIP

## Built With

* [Gatbsy](https://www.gatsbyjs.org/)
* [Emotion](https://emotion.sh/docs/introduction)
